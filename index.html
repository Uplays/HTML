<p> — párrafo

Definición: Sirve para escribir texto en un párrafo. Es la etiqueta principal para bloques de texto.

<p>Este es un párrafo de ejemplo.</p>

Este es un párrafo de ejemplo.

<style> y atributo style

Definición: <style> se usa en el <head> para CSS; el atributo style="…" aplica estilos inline directamente en una etiqueta.

<p style="color: red;">Texto en rojo</p> <p style="font-size: 20px;">Texto más grande</p> <p style="text-align: center;">Texto centrado</p>

Texto más grande

Formato de texto — <b>, <i>, <u>

Definición: etiquetas para dar formato visual al texto (negrita, cursiva, subrayado).

<b>Texto importante</b> <i>Texto en cursiva</i> <u>Texto subrayado</u>

<br> y <hr>

Definición: <br> inserta un salto de línea; <hr> dibuja una línea horizontal separadora.

<h1> a <h3> — cabeceras

Definición: se usan para títulos y subtítulos. <h1> es el más importante (grande), los números mayores son más pequeños.

<h1>Título principal</h1> <h2>Subtítulo</h2> <h3>Título más pequeño</h3>

Subtítulo

<a> — enlaces

Definición: crea un enlace. El atributo href indica la dirección. target="_blank" abre en nueva pestaña.

<a href="https://www.google.com">Ir a Google</a> <a href="https://www.google.com" target="_blank">Abrir en otra pestaña</a>

Ir a Google (nuevo tab)

<ul> / <ol> / <li> — listas

Definición: <ul> crea lista con viñetas; <ol> lista numerada; cada elemento va en <li>.

<ul> <li>Manzana</li> <li>Plátano</li> </ul> <ol> <li>Primero</li> <li>Segundo</li> </ol>
  • Manzana
  • Plátano
  1. Primero
  2. Segundo

<img> — imágenes

Definición: muestra una imagen. src = ruta; alt = texto alternativo si no carga.

<img src="foto.jpg" alt="Una foto bonita" width="300">

Ejemplo: imagen desde internet

<table> — tablas (tr, th, td)

Definición: estructura en filas y columnas. <tr> = fila, <th> = cabecera, <td> = celda.

<table border="1"> <tr> <th>Nombre</th> <th>Edad</th> </tr> <tr> <td>Juan</td> <td>20</td> </tr> </table>
NombreEdad
Juan20

<div> — divisiones / contenedores

Definición: caja genérica para agrupar contenido. Muy útil para aplicar estilos y organizar la página.

<div style="background-color: lightblue; padding: 10px;"> <h2>Título dentro del div</h2> <p>Texto dentro del div.</p> </div>

Texto dentro del div.

<form> — formularios (input, label, button)

Definición: contenedor para campos que envían datos. Incluye <input>, <label> y <button>.
